# Copilot Studio Core Concepts - Part 1

This module covers the foundational building blocks of Copilot Studio and how they come together to create a useful agent experience. Learners explore how to create an agent, define its purpose, organize conversations with topics, and bring in trusted information through knowledge sources.

The session also introduces tools as a way to move beyond static answers and connect an agent to business actions and external systems. After completing this module, participants should be able to explain the main components of a Copilot Studio agent and start assembling a well-scoped solution.

## Key Topics

- Creating agents in Copilot Studio
- Topics and conversation design
- Knowledge sources and grounding
- Using tools to extend agent capabilities
