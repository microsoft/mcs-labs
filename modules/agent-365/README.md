# Agent 365

This module provides an overview of the Agent 365 platform and the capabilities it brings to Microsoft 365 scenarios. It is intended to help learners understand the role of Agent 365, the experiences it enables, and how it complements broader Copilot and agent strategies.

Participants explore the main concepts behind the platform, the types of use cases it supports, and the considerations involved in adopting it within an organization. The goal is to establish a strong conceptual foundation before moving into more hands-on solution design.

## Key Topics

- Agent 365 platform overview
- Capabilities and core scenarios
- Relationship to Microsoft 365 experiences
- Adoption considerations
