# Orchestration and Dynamic Chaining

This module explains how orchestration helps Copilot Studio agents decide what to do next in complex interactions. Learners explore how dynamic chaining allows the platform to combine capabilities, choose tools, and coordinate multiple steps in response to user intent.

The session focuses on the concepts behind flexible agent execution, including how to design experiences that remain predictable even when the path is not fully scripted. Participants should leave with a clearer mental model for how orchestration supports more capable and adaptive agent solutions.

## Key Topics

- Orchestration concepts in Copilot Studio
- Dynamic chaining of capabilities
- Balancing flexibility and control
- Designing adaptive multi-step experiences
