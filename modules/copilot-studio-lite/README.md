# Copilot Studio Lite

This module introduces the lite experience of Copilot Studio, providing a streamlined entry point for creating AI agents. It covers the simplified interface designed for quick agent creation and basic customization without requiring deep platform knowledge.

Learners will explore how the lite experience differs from the full Copilot Studio interface and when each is appropriate for different use cases.

## Key Topics

- Copilot Studio lite experience overview
- Quick agent creation workflow
- Basic customization options
- When to use lite vs full Copilot Studio
