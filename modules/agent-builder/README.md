# M365 Agent Builder

This module introduces Agent Builder in Microsoft 365 as a fast path for creating task-focused AI assistants that work where users already spend their time. It covers the journey from simple web-grounded assistants to richer experiences that use Microsoft 365 content, including SharePoint, to provide more relevant answers and actions.

By the end of the module, learners should understand when Agent Builder is the right fit, how grounding and permissions shape the experience, and how to design assistants that are useful, trustworthy, and easy to adopt. The associated lab gives participants a chance to build and refine an M365-based agent end to end.

## Key Topics

- Agent Builder fundamentals in Microsoft 365
- Web-grounded and SharePoint-grounded assistants
- Grounding, permissions, and data access
- Designing focused assistant experiences
