# Agent Builder (SharePoint)

This module explores creating SharePoint-based AI assistants with Agent Builder so users can interact with organizational knowledge more naturally. It covers the value of grounding assistants in curated SharePoint sites, libraries, and documents while respecting the permissions model that protects enterprise content.

Learners review how content structure, searchability, and audience needs influence the quality of the final assistant. The associated lab turns those concepts into practice by guiding participants through the creation of a SharePoint-grounded assistant.

## Key Topics

- Grounding Agent Builder on SharePoint
- Content curation and site readiness
- Permissions-aware assistant design
- SharePoint-based assistant scenarios
