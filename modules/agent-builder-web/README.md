# Agent Builder (Web)

This module introduces how to create web-based AI assistants with Agent Builder and publish experiences that are grounded in web content. It focuses on the concepts behind defining scope, selecting web sources, and shaping an assistant that answers clearly and consistently for external or broadly accessible information.

The associated lab puts these ideas into practice by guiding learners through the creation of a web-grounded assistant from start to finish. After completing the module, participants should understand the design choices that improve relevance, trust, and usability in web-focused Agent Builder scenarios.

## Key Topics

- Building assistants from web content
- Selecting and grounding on public sources
- Designing focused web experiences
- Preparing for the companion hands-on lab
