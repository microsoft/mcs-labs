# Human-in-the-Loop Approvals

This module focuses on scenarios where agents should collaborate with people rather than act entirely on their own. Using expense claims as a concrete example, it explores how approval workflows combine automation with the checkpoints and accountability that many business processes require.

Learners examine approval design, routing, exception handling, and auditability so they can create solutions that are efficient without sacrificing control. The lab provides hands-on practice building an agent experience that includes human approval at the right moment.

## Key Topics

- Human-in-the-loop workflow concepts
- Approval routing and checkpoints
- Balancing automation and control
- Auditability and exception handling
