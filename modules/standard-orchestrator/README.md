# Standard Orchestrator

This module gives learners a conceptual introduction to building AI solutions with the standard orchestrator in Copilot Studio. It explains how orchestration directs requests, coordinates available capabilities, and determines the next best step across a conversation or workflow.

Rather than focusing only on mechanics, the module emphasizes when the standard orchestrator is the right choice and how to design experiences that remain understandable and maintainable. The lab lets participants apply those ideas in a concrete build scenario.

## Key Topics

- Standard orchestrator concepts
- Routing requests and capabilities
- Designing maintainable orchestrated experiences
- Applying orchestration concepts in the lab
