# Contract Alerts with Azure AI

This module introduces a proactive contract alert scenario that combines Copilot Studio concepts with Azure AI services such as Azure AI Search and Speech-to-Text. It focuses on how agents can help identify time-sensitive contract events and surface them in a way that supports timely action.

Participants learn the conceptual patterns behind proactive alerting, retrieval, and multimodal enrichment, along with the trade-offs involved in designing such a solution. The lab provides a practical setting to apply those ideas to a contract monitoring use case.

## Key Topics

- Proactive alerting scenarios
- Azure AI Search integration concepts
- Speech-to-Text and multimodal enrichment
- Designing time-sensitive agent experiences
