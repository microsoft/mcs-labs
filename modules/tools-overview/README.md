# Tools

This module provides an overview of the tool options available in Copilot Studio and how they expand what an agent can do. It covers built-in and connected capabilities such as connectors, Power Automate flows, and custom tools that let agents retrieve data, trigger processes, and integrate with business systems.

The emphasis is on choosing the right tool for the job and understanding the design trade-offs between simplicity, control, and scalability. By the end of the session, learners should be able to describe the main tool categories and identify the right extension approach for common scenarios.

## Key Topics

- Connectors and external data access
- Power Automate flows
- Custom tools and extensibility
- Choosing the right tool pattern
