# Data Fabric Agent

This module introduces the idea of a data fabric agent that helps users work across distributed information sources through a unified conversational layer. It focuses on the conceptual value of connecting diverse data locations and presenting them through a coherent agent experience.

Participants explore the importance of discovery, context, and governance when an agent spans multiple data sources. The lab that follows provides a practical setting for applying those concepts to a data fabric scenario.

## Key Topics

- Data fabric concepts for agents
- Connecting distributed information sources
- Unified conversational access
- Governance across multiple data domains
