# Copilot Studio Plugin for GitHub Copilot CLI and Claude Code

This module introduces the Copilot Studio plugin experience for developer tools such as GitHub Copilot CLI and Claude Code. It explains how specialized commands, skills, and workflows can help makers and developers author, validate, test, and manage Copilot Studio assets more efficiently.

The session is intended to bridge maker workflows and engineering workflows by showing how local tooling can speed up iteration, improve consistency, and reduce manual effort. Learners should come away understanding where the plugin fits into a modern developer experience for Copilot Studio.

## Key Topics

- Developer tooling for Copilot Studio
- CLI and skill-based workflows
- Authoring and validation support
- Bridging maker and developer practices
