# Autonomous Agents

This module focuses on autonomous agents that can operate with a higher degree of independence than traditional chat-first assistants. It introduces the concepts, capabilities, and safeguards involved when agents initiate steps, evaluate progress, and work through longer-running tasks.

Learners discuss what makes an autonomous experience trustworthy, including human oversight, tool boundaries, and measurable outcomes. The module is designed to help participants distinguish between simple automation and more capable autonomous agent behavior in business scenarios.

## Key Topics

- Autonomous agent concepts
- Long-running and goal-driven tasks
- Safety, oversight, and guardrails
- Business use cases for autonomy
