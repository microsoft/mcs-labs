## What you'll build

This lab will walk through the new **Workflows** capability:

- **Autonomous by design** — an agent that runs on its own instead of waiting for a user turn.
- **Inline agent connections** — compose and call agents directly inside a workflow.
- **Scheduled & event-driven triggers** — start work on a timer or in response to an event.
