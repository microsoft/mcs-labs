{%- comment -%} Emits a JSON manifest of all events + their resolved agenda schedules into the page. Consumed by _includes/event-nav/script.html at runtime. Schedule resolution (per event): 1. site.data.agendas[event_id].schedule if the agenda file exists. 2. Otherwise, synthesize a labs-only schedule from event.labs (each entry becomes a type=lab row). This is the fallback so authors don't need to create an agenda file for every event. Lab rows are enriched with the lab's title, URL, difficulty, and duration from site.labs so the IIFE can render the rail without additional lookups. {%- endcomment -%}
